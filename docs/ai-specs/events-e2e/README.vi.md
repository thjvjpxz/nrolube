# Event E2E Specs (Góc Nhìn Người Chơi)

Thư mục này mô tả sự kiện theo hành trình end-to-end của người chơi.

Mục tiêu:

- Giúp AI giải thích event theo ngôn ngữ gameplay dễ hiểu.
- Trả lời rõ các câu hỏi vận hành event:
  - Thu thập nguyên liệu gì.
  - Nguyên liệu lấy từ đâu.
  - Đổi nguyên liệu lấy gì.
  - Boss nào xuất hiện ở đâu.
  - Đánh boss nhận được gì.
- Không thay thế spec kỹ thuật trong `docs/ai-specs/events/`, mà bổ sung góc nhìn sản phẩm.

## Cách đọc

1. Chọn đúng event cần tìm hiểu.
2. Đọc theo thứ tự: tham gia -> nguyên liệu -> boss/map -> phần thưởng -> lưu ý trải nghiệm.
3. Nếu cần debug code, mở file spec kỹ thuật tương ứng trong `docs/ai-specs/events/`.
